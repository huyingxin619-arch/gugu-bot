# ADM龙虾看板项目

---

## 项目目标

监控 Admonitor Bot 每天的报告产出情况，方便追踪自动化运行状态。

---

## 已确认需求

| 维度 | 内容 |
|------|------|
| 展示形式 | 网页 |
| 更新频率 | 实时 |
| 时间范围 | 支持历史回溯 |
| 异常监控 | 需要（标出失败报告，告警） |
| 数据来源 | Admonitor Bot 日志（⚠️ 目前日志尚未建立，需先搭建） |

## 展示字段建议（待小胡确认）

每条报告记录建议包含：
- 客户名（脱敏）
- 报告类型
- 跑完时间
- 成功/失败状态
- 失败原因（如有）

---

用 Claude Code 写程序，小胡自己 vibe coding。
